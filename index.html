Web Projects

ForcePush


Personal


A website that utilises two REST APIs to make AJAX calls, requesting JSON data from remote servers. Using a Star Wars theme, a search function was built to do precisely that. Using IMDb's API and Star Wars' SWAPI API, both search data is parsed to display data in the same interactive area. Depending on what was loaded, themed sounds would play, such as Darth Vader's iconic breathing. Youtube's API was also used to display the latest movie trailer. Similar to Adobe Premier Pro, Sony Vegas was used to create an immersive splash page, splicing up iconic movie moments into a splash page, with a fade-in enter button.
