Tottenham News


Responsive Design


A simple website of a popular sports team, displaying the latest news and Youtube videos with social media options, but also one of my very first examples of building responsive pages by adjusting through breakpoints - that is, the point of which you shrink the site window and things get strange. Fully responsive for mobile and included an immersive splash page, which was created using the Sony Vegas video production suite once again. Rendered in a mobile friendly file size is a video of the team in training, then used as a backdrop to the splash page.
